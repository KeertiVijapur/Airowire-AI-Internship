# Tools in Data Science (TDS) – FastAPI Project Workflow & Deployment

This document summarizes the fourth session of Week 2 of the Tools in Data Science (TDS) course, which focused on building, testing, and deploying a FastAPI server along with handling requests, validation, file uploads, and live deployment.

---

## Session Details
**Course:** Tools in Data Science (TDS)

**Mode:** Recorded Online Session

---

## Session Overview
The session introduced the TDS project workflow and demonstrated how to build a complete FastAPI backend. It covered request handling, validation using Pydantic models, file uploads, automated testing, and deployment workflows. The session also showed how to containerize applications and deploy them to cloud platforms such as Hugging Face Spaces.
