# Visualizing Animated Data with PowerPoint (https://www.youtube.com/watch?v=umHlPDFVWr0)

## PowerPoint – Animated Data Visualization (Bar Chart Race)

PowerPoint can be used to create **animated data visualizations**, such as bar chart races, using transitions and manual animation techniques. It allows you to present data in a **storytelling and engaging format** without coding.

---

## What is this about?

This method focuses on:

* Creating **animated bar chart races**
* Using **slide transitions and morph effects**
* Turning static data into **dynamic visual stories**

---

## Key Features Used

* Morph transition for smooth animations
* Manual bar creation using shapes
* Slide duplication for frame-by-frame animation
* Voiceover and background audio integration
* Exporting animations as video (MP4)

---

## How It Works

### Step 1: Create Slides for Animation

* Create initial slide with bars
* Duplicate slides
* Adjust bar positions to simulate movement

---

### Step 2: Use Morph Transition

* Apply **Morph transition** between slides
* Automatically animates movement of bars

---

### Step 3: Manual Bar Creation

* Use shapes (rectangles) instead of charts
* Adjust size of bars manually based on values

---

### Step 4: Add Voiceover

* Use:

  ```
  Insert → Audio → Record Audio
  ```
* Add narration for each slide

---

### Step 5: Automate Slides

* Enable **auto-play transitions**
* Set slide duration to control animation speed

---

### Step 6: Add Background Music

* Insert audio file
* Set:

  ```
  Playback → Play across slides
  ```

---

### Step 7: Export as Video

```text
File → Export → Create a Video
```

* Save as MP4 format

---

## Example Use Case

* Bar chart race showing:

  * Movie popularity
  * Actor rankings
  * Country statistics over time
