## Flourish – Interactive & Animated Data Storytelling

Flourish is a powerful **data visualization platform** that enables you to create **interactive and animated visualizations** without coding. It is widely used for **data storytelling, dashboards, and media visualizations**.

---

## What is Flourish?

Flourish is a tool that:

* Converts data into **interactive and animated visuals**
* Supports **storytelling through transitions and motion**
* Works directly in the browser
* Allows exporting visualizations as **videos, GIFs, or embeds**

---

## Key Features

* No-code data visualization platform
* Built-in templates (Bar chart race, Line chart race, Scatter, Maps)
* Smooth animations and transitions
* Story mode for multi-step narratives
* Interactive filters and controls
* Export to MP4, GIF, or embed on web
* Custom styling and branding options

---

## Core Concept: Animation Principles

### 1. Engage + Inform

Animation should:

* Attract attention (**engage**)
* Explain data changes clearly (**inform**)

---

### 2. Object Constancy

* Keeps visual elements consistent across transitions
* Helps users track changes easily
* Improves understanding of data movement

---

## Popular Animation Templates

### Bar Chart Race

* Shows ranking changes over time
* Ideal for trends (population, revenue, rankings)

---

### Line Chart Race

* Displays time-series progression
* Smooth transitions between data points

---

### Scatter Plot Animation

* Shows relationships between variables over time
* Supports bubble size, color, and motion

---

### Morphing Charts (Story Mode)

* Transitions between chart types (e.g., line → bar → area)
* Helps explain data from multiple perspectives

---

## Key Animation Settings

### 1. Animation Duration

* Controls speed of transitions
* Measured in milliseconds or seconds

---

### 2. Stagger Animation

* Adds delay between elements
* Creates smooth, cascading motion

---

### 3. Timeline Duration

* Controls total animation time (for races)

---

### 4. Mode Duration

* Controls transitions between views

---

## Example Workflow

### Step 1: Upload Data

* Paste or upload dataset (CSV/Excel)

---

### Step 2: Choose Template

* Select chart type (e.g., bar race, scatter)

---

### Step 3: Bind Data

* Assign:

  * X-axis
  * Y-axis
  * Labels
  * Colors
  * Time field

---

### Step 4: Configure Animation

* Set duration
* Adjust speed
* Enable stagger (optional)

---

### Step 5: Create Story

* Add multiple slides
* Use transitions to guide users

---

### Step 6: Export

* Export as:

  * Video (MP4)
  * GIF
  * Embedded visualization

---
