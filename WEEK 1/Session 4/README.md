# Tools in Data Science (TDS) – Development Environment Setup & LLM CLI Tools

This document summarizes the fourth session of the Tools in Data Science (TDS) course, which focused on setting up a complete development environment on macOS and Linux systems, managing Python versions and projects, and using command-line tools to interact with Large Language Models (LLMs).

---

## Session Details
**Course:** Tools in Data Science (TDS)  
**Session:** Week 1 – Session 4  
**Mode:** Recorded Online Session  
(https://youtu.be/thACpPPTR1I)

**Focus Area:** Development environment setup, Python version management, GitHub CLI, and LLM command-line tooling  

---

## Session Overview
The session provided a comprehensive guide to setting up a professional development environment for Data Science and AI workflows. It covered installing essential tools, managing Python environments and dependencies, using GitHub directly from the terminal, and configuring command-line interfaces to work with Large Language Models for practical use cases.

---

## Topics Covered

### Agenda and Introduction
Overview of the session objectives and tools to be covered for setting up a development-ready environment.

---

### Installing Visual Studio Code
Setting up Visual Studio Code as the primary development environment on macOS and Linux systems.

---

### GitHub Education Pack & Copilot Setup
Introduction to GitHub Education benefits and configuring GitHub Copilot for assisted development.

---

### Installing Homebrew Package Manager
Installing Homebrew to manage packages and developer tools on macOS.

---

### Adding Homebrew to PATH
Configuring system environment variables to ensure Homebrew commands are accessible from the terminal.

---

### Python Version Management with pyenv
Managing multiple Python versions using pyenv and selecting appropriate versions for different projects.

---

### Installing and Using UV for Project Management
Using the UV tool to manage Python dependencies and project environments efficiently.

---

### Installing GitHub CLI
Setting up GitHub CLI to interact with GitHub repositories directly from the terminal.

---

### Creating a GitHub Repository from the Terminal
Demonstration of creating and managing GitHub repositories using command-line tools.

---

### Managing Project-Specific Python Versions
Configuring Python versions at the project level to ensure consistency across development environments.

---

### Installing the LLM Command-Line Tool
Installing tools that enable interaction with Large Language Models directly from the terminal.

---

### Configuring Gemini with the LLM Tool
Setting up Gemini API integration for LLM-based interactions.

---

### Demo: Transcribing Audio with LLM CLI
Practical demonstration of using LLM command-line tools to transcribe audio files.

### Conclusion and Final Questions
Summary of key concepts and clarification of doubts related to tools and workflows.
