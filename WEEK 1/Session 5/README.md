# Tools in Data Science (TDS) – Git & GitHub Setup with SSH

This document summarizes the fifth session of the Tools in Data Science (TDS) course, which focused on setting up Git and GitHub workflows, generating SSH keys, cloning repositories, and managing commits and pushes from the local system.

---

## Session Details
**Course:** Tools in Data Science (TDS)  
**Session:** Week 1 – Session 5  
**Mode:** Recorded Online Session  
(https://youtu.be/vBqJUhbZU3c)

**Focus Area:** Git installation, GitHub repository setup, SSH authentication, and version control workflow  

---

## Session Overview
The session provided a step-by-step walkthrough of installing Git, configuring GitHub authentication using SSH keys, creating and cloning repositories, and managing commits and pushes. Troubleshooting common Git permission and connection issues was also discussed, along with an introduction to GitHub Copilot and practical GitHub workflows.
